# Sing-box-yg: Универсальный скрипт «5-в-1» для VPS

Русская версия популярного скрипта для автоматической развертки и управления Sing-box. Поддерживает одновременную работу пяти топовых протоколов и глубокую настройку маршрутизации.

> **Важно:** Все подписки и конфигурации генерируются **локально на вашем сервере**. Скрипт не использует сторонние конвертеры или внешние API, что гарантирует полную конфиденциальность ваших данных.

---

## 🚀 Основные возможности

* **Поддержка 5 протоколов:**
* `Vless-reality-vision`
* `Vmess-ws(tls) + Argo`
* `Hysteria-2` (с поддержкой Port Hopping)
* `Tuic-v5`
* `Anytls`


* **Гибкость сетей:** Полная работа в сетях Pure IPv4, Pure IPv6 и Dual Stack.
* **Совместимость:** * **Архитектуры:** AMD64, ARM64.
* **ОС:** Ubuntu (рекомендуется), Debian, CentOS, Alpine Linux.


* **Режим для новичков:** Установка происходит максимально просто — во многих случаях достаточно нажать Enter трижды.
* **Инструменты:** Автоматическое получение сертификатов Acme, настройка туннелей Argo (временных и постоянных), уведомления в Telegram, оптимизация IP через CDN (Cloudflare).

## 🛠 Установка

Запустите следующую команду в терминале вашего VPS:

```bash
wget -N https://raw.githubusercontent.com/MyNicknme/SB_new/refs/heads/main/5protocol.sh && bash 5protocol.sh

```

### Быстрый доступ

После установки вы сможете вызывать меню управления скриптом в любое время командой:

```bash
sb

```

## 📝 Ключевые особенности настройки

1. **Маршрутизация (分流):** Возможность разделять трафик по доменам или Geosite. Например, пустить ChatGPT через WARP, а основной трафик через локальный IP сервера.
2. **Интеграция с GitLab:** Автоматическая выгрузка ваших конфигураций в приватный репозиторий GitLab для создания защищенной ссылки подписки.
3. **WARP+Socks5:** Встроенный режим проксирования через WARP или Psiphon VPN с выбором из 30+ стран для смены выходного IP.
4. **Оптимизация CDN:** Специальные настройки для протокола Vmess, позволяющие использовать «чистые» IP-адреса Cloudflare для обхода блокировок.

## 🛡 Конфиденциальность

Проект создан с упором на безопасность. В отличие от многих других скриптов, здесь:

* Нет облачных конвертеров ссылок.
* Нет сторонних логов.
* Все конфигурационные файлы SFA (Android), SFI (iOS) и SFW (Windows) создаются прямо на вашем сервере.
